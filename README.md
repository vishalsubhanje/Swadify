# 🍽️ Swadify - Food Delivery Web Application

## 📌 Project Overview

Swadify is a Java-based Food Delivery Web Application inspired by Swiggy and Zomato. It allows users to register, log in, browse restaurants, view menus, add food items to a cart, place orders, and manage their profile.

The project follows the MVC Architecture using Java Servlets, JSP, JDBC, and MySQL.

---

## 🚀 Features

- User Registration
- User Login & Logout
- Restaurant Listing
- Menu Display
- Add to Cart
- Update Cart
- Checkout
- Place Order
- Order History
- User Profile

5. Configure Apache Tomcat

6. Run the project
